# 🌐 Language Translation Tool

## 📌 Project Overview
This project is developed as part of the **CodeAlpha Internship - Task 1**.  
The **Language Translation Tool** allows users to translate text from one language to another easily using a simple and user-friendly interface.

---

## 🎯 Objective
The main goal of this project is to build a tool that:
- Takes user input text
- Allows selection of source language
- Allows selection of target language
- Displays translated output instantly

---

## 🚀 Features
- 🌍 Translate text between multiple languages
- ✍️ Simple input box for text entry
- 🔽 Dropdown for language selection
- ⚡ Fast and accurate translation
- 📱 Responsive and beginner-friendly UI

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Google Translate API / Translation Library (if used)
- VS Code
